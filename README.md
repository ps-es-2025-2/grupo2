##Resumo do Projeto Caneco Cheio(Gestor de Eventos de Bebidas)

O **Caneco Cheio (Gestor de Eventos de Bebidas** é um sistema web projetado para automatizar e padronizar a gestão de vendas, estoque e controle de caixa em eventos temporários, substituindo métodos manuais por uma solução digital.

O objetivo é reduzir erros, aumentar a eficiência operacional e garantir a rastreabilidade das operações.

---

### Versão Ideal

A **Versão 3** é a versão ideal e operacional do projeto.

| Versão | Status | Observação |
| :--- | :--- | :--- |
| **Versão 1** | Inadequada | Escopo muito pequeno em relação a um projeto real. |
| **Versão 2** | Inadequada | Apresentou falhas críticas de estrutura e instabilidade no código. |
| **Versão 3** | **Operacional** | É o resultado de uma refatoração completa, migrada para o Spring Boot, com dependências corrigidas e a API funcional. |
