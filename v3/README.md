# Justificativa da Nova Versão

Esta versão do projeto foi criada para corrigir instabilidades estruturais identificadas no código anterior.

O projeto original apresentou problemas críticos no gerenciamento de dependências e no processo de build do Maven, o que inviabilizava a execução da aplicação. Para solucionar essas inconsistências, foi realizada uma readequação completa da arquitetura.

O código atual foi migrado e ajustado para o ecossistema Spring Boot, com as dependências do Maven corrigidas. A API foi restabelecida e encontra-se funcional e operante nesta nova estrutura.
